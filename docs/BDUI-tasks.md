### BDUI-Client(клиентская библиотека для BDUI)
Задачи:
1) Реализовать sticky возможности(элементы, которые при скролле прикрепляются в началу или концу экрана), реализовать ui тесты со всеми тест-кейсами, а также обновить контракт на клиенте и на бэкенде
2) Также нужно реализовать новый экшен RemoteAction на клиенте и бэкенде, по сути это как ForwardAction, но вместо верстки возвращает список экшенов. Кейс для использования: RemoteAction используется на клиенте в нестандартных сценариях, н-р, при входе в приложение будет RemoteAction на определенную ручку с параметрами, в ответ придет н-р ForwardAction на главный экран, если успешная авторизация. Также напиши тесты для этого
3) Проанализировать виды переменных, добавить возможность отправлять переменные при каждом запросе экрана. То есть некоторые переменные будут отправляться в body request со значениями. Также реализовать удобное api на серверной части, эти данные должны автоматически парситься и храниться в контексте запроса. Написать клиентские и бэкендные тесты.
